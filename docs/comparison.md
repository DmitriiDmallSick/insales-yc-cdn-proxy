# Сравнение скорости: с YC CDN прокси vs без оптимизации

Все замеры сделаны вручную через DevTools → Save as HAR. Холодная загрузка (Disable cache), десктоп, один и тот же провайдер и момент времени для каждого набора измерений.

## Методология

Все HAR сняты вручную в Chrome DevTools с опцией **Disable cache** на одном устройстве и одном соединении для сравниваемых сайтов. Замеры в разные дни могут отличаться из-за нестабильности блокировки РКН - CDN InSales то блокируется, то частично работает в зависимости от момента времени и маршрутизации конкретного провайдера.

Для воспроизводимого тестирования с симуляцией блокировки можно использовать `iptables` для блокировки IP `87.242.124.98` на уровне контейнера - тогда результаты стабильны и не зависят от текущего состояния сети.
